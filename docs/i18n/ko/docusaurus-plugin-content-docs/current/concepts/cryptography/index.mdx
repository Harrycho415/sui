암호 민첩성은 Sui의 핵심이다. 시스템은 여러 암호 알고리즘과 프리미티브를 지원하며, 이들 사이를 빠르게 전환할 수 있다. Sui를 사용하면 시스템에 적합한 암호 솔루션을 선택하고, 최신 알고리즘을 사용할 수 있게 되면 이를 구현할 수 있다.

Sui는 공개 키, 서명, 집계된 서명, 해시 함수와 같은 암호 프리미티브를 전체 저장소에서 공유되는 하나의 통합된 type alias 또는 enum wrapper 아래에 정의한다. 이러한 프리미티브를 변경하면 애플리케이션의 모든 구성 요소에 영향을 준다. 애플리케이션의 암호 구성을 빠르게 업데이트하면서 일관된 보안을 유지할 수 있다.
