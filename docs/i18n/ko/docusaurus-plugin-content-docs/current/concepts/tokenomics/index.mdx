Tokenomics라는 용어는 토큰과 economics라는 두 단어의 조합이다. 이 용어는 일반적으로 블록체인의 경제 원리와 동작을 설명하는 데 사용된다.

블록체인에는 종종 네트워크의 통화 역할을 하는 native 토큰이 있다. Native 토큰은 transactions와 compute 및 storage 같은 리소스 사용 비용을 지불하는 데 사용된다.
