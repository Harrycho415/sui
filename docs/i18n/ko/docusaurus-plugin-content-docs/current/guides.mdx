이 섹션은 Sui 개발 여정을 가속하도록 설계된 실용적이고 구현 중심의 예시를 제공한다.

블록체인 개발이 처음이든 숙련된 Sui 개발자이든, 이 자료는 애플리케이션을 구축하고 자산을 생성하며 Sui의 고유한 기술적 기능을 효과적으로 활용할 수 있도록 구조화된 안내를 제공한다.

도구를 설치하고 환경을 설정한 뒤 "Hello, World!" Move 패키지를 배포하여 Sui 개발 경험을 시작한다.
