이 섹션에서 계속 늘어나는 예제들은 Sui 블록체인을 위한 패키지를 보여준다. Sui 및 Move 전문가들이 작성했으므로, 이 예제들에 사용된 기술을 추출하여 자신의 Sui 프로젝트에 적용한다.

<ImportContent source="warn-ml-rpcs.mdx" mode="snippet" />

## Examples

Sui는 Sui 블록체인에 적합한 프로그래밍 기술을 안내하기 위해 다양한 예제를 제공하는 데 전념하고 있다. 이 목록은 계속 늘어날 것이므로, 자주 확인한다.

:::tip

프로젝트는 스택 유형별로 그룹화되어 있으며 복잡성 순으로 정렬된다.

:::

### Full-stack apps

- [Distributed Counter](./app-examples/e2e-counter.mdx):  누구나 증가시킬 수 있지만 object 소유자만 재설정할 수 있는 기본적인 탈중앙화 카운터를 생성하는 엔드투엔드 예제이다. 이 예제는 패키지를 생성하는 Move 코드를 포함하며 기본 프론트엔드를 제공하기 위해 Sui TypeScript SDK를 활용한다.
- [Trustless Swap](./app-examples/trustless-swap.mdx): 이 예제는 shared object를 에스크로 계정으로 사용하여 Sui 블록체인에서 무신뢰 스왑을 보여준다.
- [Coin Flip](./app-examples/coin-flip.mdx): Coin Flip 앱은 온체인 무작위성을 보여준다.
- [Reviews Rating](./app-examples/reviews-rating.mdx): 이 예제는 Sui에서 식품 서비스 산업을 위한 Reviews Rating 플랫폼을 구현하는 것을 보여준다.
- [Blackjack](./app-examples/blackjack.mdx): 이 예제는 인기 있는 카지노 카드 게임인 Blackjack의 온체인 버전 이면의 로직을 보여준다.
- [Plinko](./app-examples/plinko.mdx): 이 예제는 클래식 Plinko 게임을 온체인으로 구현하고 공정하고 투명한 확률 게임을 만들기 위해 암호화 기반 전략을 사용하는 것을 보여준다.

### Smart contracts

- [Tic-tac-toe](./app-examples/tic-tac-toe.mdx): 사용자 상호 작용에 대한 다양한 접근 방식을 보여주기 위한 Sui 네트워크의 클래식 tic-tac-toe 게임의 세 가지 구현이다.

### Smart contracts &amp; Backend

- [Weather Oracle](./app-examples/weather-oracle.mdx): Sui Weather Oracle은 실시간 날씨 데이터를 제공하는 기본 날씨 오라클을 생성하는 방법을 보여준다.
