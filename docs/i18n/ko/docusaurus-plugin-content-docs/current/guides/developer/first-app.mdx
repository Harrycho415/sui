Sui 네트워크에서 첫 번째 dApp을 생성하기 전에 [Sui installed](./getting-started/sui-install.mdx) ￼되어 있어야 한다.

dApp은 *decentralized application*의 약어로, 중앙 서버 대신 블록체인 또는 탈중앙화 네트워크에서 실행되는 애플리케이션이다. 일반적인 dApp은 블록체인에서만 실행되는 것이 아니라, 블록체인에 존재하는 코드와 상호작용하는 TypeScript 프론트엔드와 같은 여러 구성 요소로 이루어진다. Sui에서는 온체인에 존재하는 dApp의 구성 요소를 위해 Move로 작성된 코드가 필요하다. 이러한 구성 요소는 패키지, 모듈, 또는 smart contracts라고 불린다.

이 섹션의 지침은 기본 패키지 작성, 코드 디버깅 및 테스트, 그리고 게시 과정을 안내한다. 이 실습을 완료하려면 지침이 나타나는 순서대로 따라야 한다.

일부 지침에서는 `move` Sui CLI 명령을 사용한다. Sui CLI는 바이너리와 함께 설치되므로, 설치 지침을 따랐다면 시스템에 이미 설치되어 있다. 설치 여부를 확인하려면 터미널 또는 콘솔에서 다음 명령을 실행한다.

```sh
$ sui --version
```

콘솔이 다음과 유사한 버전 번호를 반환하지 않는 경우, Sui 설치 지침을 참조한다.

```sh
$ sui 1.xx.x-abc123xyz
```

## Connecting to a network

Sui를 설치한 후 네트워크에 연결할 수 있다. Sui에는 세 개의 퍼블릭 네트워크(Devnet, Testnet, Mainnet)가 있으며, 로컬 Sui 네트워크를 실행하고 연결할 수도 있다. 각 네트워크마다 해당 네트워크에 특화된 온체인 address가 필요하다. address는 `0x8bd4613c004aac53d06bb7ceb7f46832c9ae69bdc105dfc5fcac225d2061fcac` 형식의 고유 ID를 가진 object이다.<br>이 address 외에도 패키지 게시 및 해당 패키지에 대한 Move 호출과 같은 온체인 활동과 관련된 가스 수수료를 지불하기 위해 SUI가 필요하다. Mainnet을 제외한 모든 네트워크에서는 패키지 개발을 위해 계정에 무료 SUI 코인을 받을 수 있다. 이 예제에서는 Testnet 네트워크에 연결한다.

### Connect to Testnet

이미 네트워크 설정이 되어 있다면 활성 환경을 Testnet으로 전환한다.<br>다음 지침은 초기 설정을 위한 것이다.

1. 터미널 또는 콘솔에서 다음 명령을 사용하여 구성을 시작한다:
    ```sh
    $ sui client
    ```
2. 프롬프트에서 <kbd>y</kbd>를 입력하고 <kbd>Enter</kbd> 를 눌러 Sui 풀 노드 서버에 연결한다
3. 다음 프롬프트에서 Testnet 서버의 address(`https://fullnode.testnet.sui.io:443`)를 입력하고 <kbd>Enter</kbd> 를 누른다.
4. 다음 프롬프트에서 네트워크에 별칭을 부여하기 위해 `testnet`을 입력하고 <kbd>Enter</kbd> 를 누른다. 이후 명령에서 전체 URL 대신 이 별칭을 사용할 수 있다.
5. 다음 프롬프트에서 `0`을 입력하고 <kbd>Enter</kbd> 를 누른다. 선택하면 `ed25519` 서명 방식에서 address가 생성된다.
6. 응답에는 address의 별칭, 실제 address ID, 그리고 비밀 복구 구문이 제공된다. 이후 참조를 위해 이 정보를 반드시 저장한다. Testnet 네트워크이므로 Mainnet일 때만큼 이 정보의 보안이 중요하지는 않다.
7. 터미널 또는 콘솔에서 다음 명령을 사용하여 계정에 SUI를 받는다.
    ```sh
    $ sui client faucet
    ```
8. 다음 명령을 사용하여 SUI를 수령했는지 확인할 수 있다. 네트워크 활동에 따라 코인을 받는 데 지연이 발생할 수 있다.
    ```sh
    $ sui client gas
    ```

이제 Sui Testnet 네트워크에 연결되었으며, 사용 가능한 SUI를 보유한 계정을 가지고 있어야 한다.
