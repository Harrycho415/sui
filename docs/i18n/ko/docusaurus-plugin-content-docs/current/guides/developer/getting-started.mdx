Sui는 최초의 인터넷 규모 프로그래밍 가능 블록체인 플랫폼이다. 언뜻 보면 마케팅 문구처럼 들릴 수 있지만, 관련 문서를 살펴보면 Sui가 블록체인의 대중화를 가로막는 여러 문제점을 해결하고 있음을 알게 될 것이다.

Sui에서 개발을 시작하기에 앞서 코드 저장소를 이해하고 바이너리를 설치를 해야한다.

## Move IDEs and plugins

다음 표는 Move 확장 기능을 지원하는 검증된 IDE 목록이다. 이러한 IDE와 플러그인은 필수 사항은 아니지만 Sui에서 개발할 때 더 나은 환경을 제공한다.

IDE | Move plugin | Description
--- | --- | ---
[Visual Studio Code](https://code.visualstudio.com/) | • [Move extension](https://marketplace.visualstudio.com/items?itemName=mysten.move)<br>• [Move syntax](https://marketplace.visualstudio.com/items?itemName=damirka.move-syntax) | Microsoft에서 Move 개발을 위해 권장하는 무료 IDE이다.
[IntelliJ IDEA](https://www.jetbrains.com/idea/) | [Move Language Plugin](https://plugins.jetbrains.com/plugin/14721-move-language) | JetBrains에서 개발한 상용 IDE이다.
[Emacs](https://www.gnu.org/software/emacs/) | [move-mode](https://github.com/amnn/move-mode) | 무료 오픈소스 텍스트 편집기이다.
[Github Codespaces](https://github.com/features/codespaces) | [Move Syntax](https://marketplace.visualstudio.com/items?itemName=damirka.move-syntax) | GitHub에서 제공하는 웹 기반 IDE이다.

:::info

일부 플러그인은 [Move 2024](./advanced/move-2024-migration.mdx)에 대한 하이라이팅 기능을 아직 지원하지 않을 수 있다.

:::
