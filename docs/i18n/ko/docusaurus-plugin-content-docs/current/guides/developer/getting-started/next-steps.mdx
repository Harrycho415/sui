Getting Started 경로를 따라왔다면 Sui에서 여정을 계속할 준비가 되었어야 한다.

이 페이지는 학습을 계속하는 방법, 추가 리소스, developer 커뮤니티에 참여하는 방법을 설명한다.

<details>

<summary>Verify "Getting Started" essentials</summary>

필요한 것을 모두 갖추었는지 확인하려면 다음 명령을 로컬 환경의 콘솔에 붙여 넣는다:

```sh
$ sui --version
$ sui client active-address
$ sui client active-env
$ sui client gas
```

모든 명령이 성공하면 콘솔에 다음과 유사한 출력이 표시된다:

```sh
sui [version_number]-[commit_hash]
0x6543...7241
testnet
╭───────────────┬────────────────────┬──────────────────╮
│ gasCoinId     │ mistBalance (MIST) │ suiBalance (SUI) │
├───────────────┼────────────────────┼──────────────────┤
│ 0x1e86...63b2 │ 5041161576         │ 5.04             │
╰───────────────┴────────────────────┴──────────────────╯
```

오류가 발생하면 오류를 일으켰거나 출력이 없었던 명령과 관련된 내용을 다시 확인한다.

| Command | Successful if |
| ------- | -------------- |
| `sui --version` | [installed Sui](/guides/developer/getting-started/sui-install.mdx)를 올바르게 수행했다. |
| `sui client active-address` | 현재 활성화된 [Sui address](/guides/developer/getting-started/get-address.mdx)가 있다. |
| `sui client active-env` | 현재 활성화된 [connection to Testnet](/guides/developer/getting-started/configure-sui-client.mdx)을 설정했다. |
| `sui client objects` | 현재 address가 [SUI for testing](/guides/developer/getting-started/get-coins.mdx)을 소유한다. |

</details>

## Additional documentation

이 사이트의 나머지 문서에는 Sui 여정을 계속하는 데 중요한 개념, 가이드, 예제가 들어 있다.

또한 이 문서 밖에도 참고할 수 있는 리소스가 있으며, Sui를 최대한 활용하도록 경험과 지식을 활용할 수 있는 크고 계속 성장하는 커뮤니티도 있다.

### SDK documentation

Sui 생성을 시작한 회사인 Mysten Labs는 네트워크와 관련 자산과 상호작용하기 위한 SDK를 제공한다.

자세한 내용은 [SDK docs](https://sdk.mystenlabs.com/typescript)를 방문한다.

### The Move Book

Move는 Sui의 smart contract 언어이다.

[The Move Book](https://move-book.com/)은 이 언어를 자세히 설명한다.

## Connect with the Sui community

- [Sui Discord](https://discord.gg/sui)에 참여한다.
- [Telegram](https://web.telegram.org/z/?ref=blog.sui.io#-1742460518)에서 **Suinami Riders**에 참여한다.
- [Sui Developer Forum](https://forums.sui.io/)을 확인한다.
- X에서 [@SuiFoundation](https://twitter.com/@SuiFoundation)을 팔로우한다.

### Schedule office hours

Sui 팀으로부터 developer 지원이나 조언을 받기 위한 1:1 세션에 [Sign up](https://cal.com/forms/08983b87-8001-4df6-896a-0d7b60acfd79)한다.

### Awesome Sui

[Awesome Sui](/references/awesome-sui.mdx) repo는 Sui 생태계의 developer 도구와 인프라 project를 선별해 정리한 목록이다.

### Upcoming hackathons

다른 developer와 함께 애플리케이션을 빌드하려면 [upcoming hackathon events](https://sui.io/ecosystem-hub)를 확인한다.

### Developer newsletter

Sui에서 빌드하는 데 도움이 되는 집중도 높은 콘텐츠를 받으려면 [Sui Developer Newsletter](https://developers.sui.io/)에 등록한다.

### Developer portal

코스, 비디오, 기타 학습 리소스로 연결되는 링크를 찾으려면 [Sui Developer Portal](https://sui.io/developers)을 방문한다.
