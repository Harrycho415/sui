[Obtain a Sui Address](/guides/developer/getting-started/get-address)에서 사용한 CLI 옵션과 달리, Sui 네트워크에 연결되는 wallet은 일부 사용자에게 더 적합할 수 있는 시각적 workflow를 제공한다.

다음 Chrome extension wallet은 브라우저와 통합되며 address 생성 과정을 안내하는 UI workflow를 제공한다.

Address를 얻는 정확한 절차는 선택한 extension의 문서를 참조한다.

:::info

이 extension에 대한 링크는 편의를 위해 제공된다.

이 링크가 포함되었다고 해서 어떤 종류의 보증, 권장, 승인도 의미하지 않는다.

어떤 extension을 사용할지 결정하기 전에 스스로 충분히 검토한다.

:::

- [Slush — A Sui wallet](https://chromewebstore.google.com/detail/sui-wallet/opcgpfmipidbgpenhmajoajpbobppdil): 이전에는 Sui Wallet로 알려졌으며, Slush는 Sui를 만든 팀인 Mysten Labs가 만든 wallet이다.
- [Suiet | Sui Wallet](https://chromewebstore.google.com/detail/suiet-sui-wallet/khpkpbbcccdmmclmpigdgddabeilkdpd): Suiet wallet은 Sui 블록체인 위에 구축된, 모두를 위한 wallet이다.
- [Glass wallet](https://chromewebstore.google.com/detail/glass-wallet-sui-wallet/loinekcabhlmhjjbocijdoimmejangoa): Glass Wallet은 코인, 키, NFT를 안전하게 제어할 수 있는 non-custodial wallet이다.
- [Martian Aptos & Sui Wallet Extension](https://chromewebstore.google.com/detail/martian-aptos-sui-wallet/efbglgofoippbgcjepnhiblaibcnclgk): Martian은 Aptos와 Sui를 위한 self-custodial crypto wallet이다.
- [Surf Wallet](https://chromewebstore.google.com/detail/surf-wallet/emeeapjkbcbpbpgaagfchmcgglmebnen): Surf Wallet은 블록체인에서 token을 편리하고 안전하게 전송하고 NFT를 수집할 수 있게 하는 플랫폼이다.
- 사용 가능한 Sui wallet extension의 전체 목록은 [chrome web store](https://chromewebstore.google.com/search/sui%20wallet)를 방문한다.
