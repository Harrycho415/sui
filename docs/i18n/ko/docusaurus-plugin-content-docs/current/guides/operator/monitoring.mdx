:::info

이 지침은 고급 사용자를 위한 것이다. 로컬 개발 환경만 필요하다면, 로컬 풀 노드, validator, faucet을 생성하려면 [Create a Local Sui Network](../developer/getting-started/local-network.mdx)의 지침을 따라야 한다.

:::

노드는 기본적으로 `localhost:9184/metrics`에서 노출한다.

메트릭 UI에서 메트릭을 확인할 수 있고, `curl`과 같은 도구를 사용해 파싱하기 쉬운 형식으로 메트릭을 가져올 수도 있다.

```sh
$ curl -s http://localhost:9184/metrics | grep -E 'sui_validator|sui_fullnode'
```

## Production monitoring

프로덕션 모니터링에는 [Prometheus](https://prometheus.io/)와 [Grafana](https://grafana.com/)를 사용하는 것이 권장된다.

노드에서 메트릭을 수집하기 위해 Grafana Agent, Grafana Alloy 또는 다른 도구를 사용할 수 있다.
