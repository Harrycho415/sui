해당 가이드는 SuiPlay0X1 휴대용 게임기에 개발을 하는 게임 개발자를 위한 개요와 권장 사항을 제공한다. [SuiPlay0X1](https://www.suiplay0x1.com/)은 Web3를 지원하는 최초의 휴대용 게임 콘솔이다. 다양한 PC 게임 뿐만 아니라 Sui 기술을 사용하여 개발된 최신 AAA 게임도 지원한다.

자산이 개별 게임 내에 한정되어있는 기존 게임과 달리, Web3 게임은 블록체인 지갑을 통해 진정한 디지털 소유권을 보장한다. 플레이어는 다양한 게임과 플랫폼에서 게임 내 자산을 소유, 거래 및 전송할 수 있다. SuiPlay0X1에서 지갑 통합은 이처럼 향상된 게임 경험을 제공하는 핵심적인 역할을 한다.
