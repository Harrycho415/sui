이 가이드는 [SuiPlay0X1](https://www.suiplay0x1.com/) 휴대용 게임기용으로 개발하는 게임 개발자를 위한 개요와 권장 사항을 제공한다. SuiPlay0X1은 Web3를 지원하는 최초의 휴대용 게임 콘솔이다. 다양한 PC 게임뿐 아니라 Sui 기술을 사용해 개발된 새로운 AAA 타이틀도 지원한다.

개별 게임 안에 자산이 묶여 있는 전통적인 게임과 달리, Web3 게임은 블록체인 지갑을 통해 진정한 디지털 소유권을 가능하게 한다. 플레이어는 서로 다른 게임과 플랫폼 전반에서 게임 내 자산을 소유하고, 거래하고, 전송할 수 있다. SuiPlay0X1에서는 지갑 통합이 이러한 향상된 게임 경험을 제공하는 핵심 요소이다.
