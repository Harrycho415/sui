이 페이지에서는 Sui에 기여하는 방법을 설명하고 Sui 커뮤니티 참여에 대한 추가 정보를 제공한다.

## Join the community {#join-the-community}

Sui 커뮤니티와 소통하려면 [Discord](https://discord.gg/sui) 에 가입한다.

## Open issues {#open-issues}

Sui 관련 문제를 보고하려면, GitHub 저장소에 [create an issue](https://github.com/MystenLabs/sui/issues/new/choose) 를 한다. **'Get started'** 를 클릭하여 생성할 이슈 유형에 대한 템플릿을 연다.

## Fork Sui to contribute {#fork-sui-contribute}

Sui 소스 코드 또는 문서에 기여하려면 GitHub 계정만 있으면 된다. GitHub 웹사이트에서 직접 업데이트를 커밋하고 PR을 제출하거나, 저장소를 포크하여 로컬 환경에서 원하는 도구를 사용하여 변경할 수 있다. PR은 항상 `main` 브랜치에 제출해야 한다.

필요한 경우, Sui 저장소를 포크하는 방법에 대한 지침은 [Sui Environment Setup](./sui-environment.mdx) 참조한다

## Contribute via the Sui Improvement Proposal (SIP) process {#SIP}

Sui 네트워크는 오픈 소스, 탈중앙화, 무허가형 프로토콜로, 커뮤니티의 참여를 환영한다. 핵심 프로토콜 업그레이드에 대한 아이디어가 있다면 Sui 개선 제안(SIP)을 제출하는 것이 가장 좋은 방법이다. SIP에 대한 자세한 내용은 [Contribute to Sui through SIPs](https://blog.sui.io/sui-improvement-proposals-sips/)를 참조한다.
