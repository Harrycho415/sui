Sui RPC용 GraphQL은 Sui [network](https://sui.io/networkinfo)와 상호작용할 수 있게 해주는 공개 서비스이다.

Sui RPC용 GraphQL을 시작하려면 [Getting Started](/concepts/data-access/graphql-rpc) 가이드를 확인한다.

GraphQL 서비스에서 사용되는 개념을 더 알아보려면 Sui RPC 개념 페이지의 [GraphQL](/concepts/data-access/graphql-rpc.mdx)을 확인한다.

참조 문서를 보려면 [schema](./sui-api/sui-graphql/beta/reference)를 확인한다.

## Key types

모든 GraphQL API 요소는 왼쪽 사이드바를 통해 접근할 수 있으며, 다음은 탐색을 시작하기에 좋은 출발점이다.

- "Queries"는 체인 상태를 읽기 위한 모든 최상위 쿼리를 나열하며, address와 object에 대한 세부 정보를 읽는 것부터 <UnsafeLink href="/references/sui-api/sui-graphql/beta/reference/operations/queries/simulate-transaction.mdx">dryRunTransactionBlock</UnsafeLink>까지 포함하고, 이는 실행과 유사한 인터페이스를 가지지만 체인을 수정하지 않는다.
- <UnsafeLink href="/references/sui-api/sui-graphql/alpha/reference/types/objects/object">Object</UnsafeLink>는 모든 온체인 object(Move 값과 패키지)를 나타내는 type이다.
- <UnsafeLink href="/references/sui-api/sui-graphql/alpha/reference/types/objects/address">Address</UnsafeLink>는 계정 address에 해당하며, 이는 transaction에 서명하는 서명의 공개 키에서 파생되고, 이 계정이 소유한 object와 서명했거나 영향을 받은 transaction을 질의하는 데 사용할 수 있다.
<UnsafeLink href="/references/sui-api/sui-graphql/beta/reference/types/objects/owned-or-immutable">Owner</UnsafeLink>는 <UnsafeLink href="/references/sui-api/sui-graphql/beta/reference/types/objects/move-object">MoveObject</UnsafeLink>를 소유할 수 있는 모든 entity를 나타내며, 소유자가 <UnsafeLink href="/references/sui-api/sui-graphql/beta/reference/types/objects/object">Object</UnsafeLink>인지 <UnsafeLink href="/references/sui-api/sui-graphql/beta/reference/types/objects/address">Address</UnsafeLink>인지 알 수 없는 경우를 처리한다.

예를 들면 Move object의 관점에서 자신의 owner를 볼 때가 여기에 해당한다.
