
```sh
Sui 네트워크와 상호작용하기 위한 클라이언트

Usage: sui client [OPTIONS] [COMMAND]

Commands:
  active-address              지정하지 않았을 때 명령에 사용되는 기본 address
  active-env                  지정하지 않았을 때 명령에 사용되는 기본 환경
  addresses                   클라이언트가 관리하는 Addresses를 가져온다
  balance                     address의 코인 잔액을 나열한다
  call                        Move 함수를 호출한다
  chain-identifier            rpc endpoint에서 chain identifier를 조회한다
  dynamic-field               address로 dynamic field를 조회한다
  envs                        모든 Sui 환경을 나열한다
  execute-signed-tx           서명된 Transaction을 실행한다. 사용자가 다른 곳에서 서명하고
                                  이 명령으로 실행할 때 유용하다
  execute-combined-signed-tx  결합된 직렬화 SenderSignedData 문자열을 실행한다
  faucet                      faucet에서 가스 코인을 요청한다. 기본적으로 active
                                  address와 active 네트워크를 사용한다
  gas                         address가 소유한 모든 gas object를 가져온다. address 대신
                                  address의 alias를 사용할 수 있다
  merge-coin                  두 개의 coin object를 하나의 coin으로 병합한다
  new-address                 키페어 체계 플래그 {ed25519 | secp256k1 | secp256r1}로 새
                                  address와 키페어를 생성하며 선택적 파생 경로를 사용한다. 기본값은
                                  ed25519의 경우 m/44'/784'/0'/0'/0', secp256k1의 경우 m/54'/784'/0'/0/0,
                                  secp256r1의 경우 m/74'/784'/0'/0/0이다. 단어 길이는 { word12 |
                                  word15 | word18 | word21 | word24}를 사용할 수 있으며 지정하지 않으면
                                  기본값은 word12이다
  new-env                     새 Sui 환경을 추가한다
  object                      object 정보를 가져온다
  objects                     address가 소유한 모든 object를 가져온다. alias 형태의
                                  address도 허용한다
  party-transfer              object를 party ownership으로 전송한다
  pay                         입력 코인으로 지정된 금액에 따라 수신자에게 코인을 지불한다.
                                  recipients 길이는 amounts 길이와 같아야 한다
  pay-all-sui                 입력 코인으로 가스 비용을 차감한 뒤 남은 모든 SUI 코인을
                                  수신자에게 지불한다. 입력 코인에는 가스 지불용 코인도 포함되므로
                                  추가 가스 코인이 필요하지 않다
  pay-sui                     입력 코인으로 지정된 금액에 따라 수신자에게 SUI 코인을
                                  지불한다. recipients 길이는 amounts 길이와 같아야 한다. 입력 코인에는 가스
                                  지불용 코인도 포함되므로 추가 가스 코인이 필요하지 않다
  ptb                         제공된 인자로 PTB를 실행한다
  publish                     Move 모듈을 게시한다
  serialized-tx               이미 직렬화된 transaction을 실행, 드라이 런, dev-inspect
                                  또는 기타 방식으로 검사한다
  serialized-tx-kind          이미 직렬화된 transaction kind를 실행, 드라이 런, dev-inspect
                                  또는 기타 방식으로 검사한다
  split-coin                  coin object를 여러 coin으로 분할한다
  switch                      활성 address와 네트워크(예: devnet, local rpc server)를 전환한다
  tx-block                    지정된 transaction block 실행의 effects를 가져온다
  transfer                    object를 전송한다
  transfer-sui                SUI를 전송하고 같은 SUI coin object로 가스를 지불한다. amount가
                                  지정되면 해당 amount만 전송되고, 그렇지 않으면 전체 object가 전송된다
  upgrade                     Move 모듈을 업그레이드한다
  verify-bytecode-meter       패키지에 대해 bytecode verifier를 실행한다
  verify-source               로컬 Move 패키지를 온체인 패키지와 비교 검증하며,
                                  선택적으로 해당 의존성도 검증한다
  remove-address              alias 또는 hexadecimal 문자열로 기존 address를 제거한다
  replay-transaction          transaction effects를 보기 위해 지정된 transaction을 리플레이한다.
                                  디버그하려면 환경 변수 MOVE_VM_STEP=1을 설정한다
  replay-batch                파일에 나열된 transaction들을 리플레이한다
  help                        이 메시지 또는 지정된 하위 명령의 도움말을 출력한다

Options:
      --client.config <CONFIG>  사용자 계정 상태를 저장하는 파일을 설정한다(없으면 빈 파일을
                                생성한다)
      --client.env <ENV>        사용할 Sui 환경이다. 현재 config 파일에 존재해야 한다
      --json                    명령 출력값을 json 형식으로 반환한다
  -y, --yes                     
  -h, --help                    도움말을 출력한다
  -V, --version                 버전을 출력한다
```
