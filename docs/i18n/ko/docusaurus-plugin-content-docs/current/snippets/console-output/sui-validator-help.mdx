
```sh
validator 및 validator 후보를 위한 도구

Usage: sui validator [OPTIONS] [COMMAND]

Commands:
  make-validator-info               
  become-candidate                  
  join-committee                    
  leave-committee                   
  display-metadata                  
  update-metadata                   
  update-gas-price                  Reference Gas Price 계산에 사용되는 가스 가격을 갱신한다
  report-validator                  validator를 신고하거나 신고 취소한다
  serialize-payload-pop             Proof of Possession 생성에 사용되는 payload를 직렬화한다.
                                        이 기능은 Authority protocol keypair가 서명할 수 있도록
                                        payload를 오프라인으로 가져갈 때 유용하다
  display-gas-price-update-raw-txn  validator의 가스 가격 quote를 설정하는 transaction의
                                        직렬화 데이터를 출력한다
  register-bridge-committee         Sui native bridge committee 멤버 등록
  update-bridge-committee-node-url  sui native bridge committee node url 갱신
  help                              이 메시지 또는 지정된 하위 명령의 도움말을 출력한다

Options:
      --client.config <CONFIG>  사용자 계정 상태를 저장하는 파일을 설정한다(없으면 빈 파일을
                                생성한다)
      --json                    명령 출력값을 json 형식으로 반환한다
  -y, --yes                     
  -h, --help                    도움말을 출력한다
  -V, --version                 버전을 출력한다
```
