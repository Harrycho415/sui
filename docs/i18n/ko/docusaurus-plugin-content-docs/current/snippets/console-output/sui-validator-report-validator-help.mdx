
```sh
validator를 신고하거나 신고를 취소한다

Usage: sui validator report-validator [OPTIONS] <reportee-address>

Arguments:
  <reportee-address>  신고 또는 신고 취소 대상 validator의 Sui Address이다

Options:
      --operation-cap-id <operation-cap-id>
          sender가 reporter validator 자신이고 Cap object를 보유한 경우에는 선택 사항이다.
          sender가 reporter validator 자신이 아닌 경우에는 필수이다. validator의
          OperationCap ID는 `display-metadata` 하위 명령으로 찾을 수 있다
      --undo-report <undo-report>
          true이면 기존 신고를 취소한다 [possible values: true, false]
      --gas-budget <gas-budget>
          이 transaction의 가스 예산이다
      --json
          명령 출력값을 json 형식으로 반환한다
  -h, --help
          도움말을 출력한다
  -V, --version
          버전을 출력한다
```
