`Processor`는 동시 처리 엔진으로, 최대 처리량을 위해 여러 작업을 동시에 처리한다. 프로세서의 주요 역할은 병렬 작업자를 사용하여 primary 체크포인트 데이터를 데이터베이스에 바로 사용할 수 있는 행으로 변환하는 것이다.

이 구성 요소는 병렬 처리를 위해 `FANOUT` 워커 태스크(기본값: 10)를 생성하여 이 작업을 처리한다. `FANOUT` 은 병렬 처리 용량을 제어하는 ​​핵심 구성 요소이다.
