:::info

패키지 게시 방법에 대한 자세한 내용은 [Publish a Package](/guides/developer/first-app/publish.mdx) 가이드를 참조하거나 Sui CLI에서 사용할 수 있는 `client` 명령 전체 목록은 [Sui Client CLI](/references/cli/client.mdx) 를 참조한다.

:::

코드를 퍼블리시하기 전에, 아직 Sui Client CLI를 초기화하지 않았다면 먼저 초기화해야 한다. 초기화하려면 프로젝트의 루트 디렉터리에서 터미널이나 콘솔을 열고 `sui client` 입력한다. 다음과 같은 응답이 표시되면 나머지 지침을 완료한다:

```
Config file ["<FILE-PATH>/.sui/sui_config/client.yaml"] doesn't exist, do you want to connect to a Sui full node server [y/N]?
```

Enter `y` to proceed. You receive the following response:

```
Sui full node server URL (Defaults to Sui Testnet if not specified) :
```

이 칸을 비워 둔다(Enter 키를 누름). 다음과 같은 응답이 표시된다:

```
Select key scheme to generate key pair (0 for ed25519, 1 for secp256k1, 2: for secp256r1):
```

Select `0`. Now you should have a Sui address set up.
