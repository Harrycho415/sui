**JSON-RPC is deprecated**. 2026년 7월까지 [gRPC](/concepts/data-access/grpc) 또는 [GraphQL RPC](/concepts/data-access/custom-indexers.mdx)로 마이그레이션하라.

풀 노드에서 gRPC를 활성화했거나 GraphQL RPC를 제공하는 [RPC 또는 데이터 제공자 목록](https://www.notion.so/mystenlabs/RPC-providers-offering-future-Sui-data-primitives-2466d9dcb4e980a99a36e9aafd8c17e0?source=copy_link)을 참조하라.
접근 권한을 요청하려면 제공자에 직접 문의하라.
RPC 또는 데이터 제공자가 아직 이러한 데이터 접근 방법을 지원하지 않는다면 지원 활성화를 요청하거나 도움을 받기 위해 Discord, Telegram, 또는 Slack에서 Sui Foundation 팀에 문의하라.
