Move는 온체인 objects( *smart contracts* 이라고도 함)를 조작하는 안전한 패키지를 작성하기 위한 오픈 소스 언어이다. Move는 플랫폼에 구애받지 않는 언어로, 데이터 및 실행 모델이 매우 다른 블록체인 전반에 걸쳐 공통 라이브러리, 도구 및 개발자 커뮤니티를 구축할 수 있도록 지원한다. Move는 코드가 실행되는 블록체인의 요구 사항에 맞게 조정할 수 있다. Sui 블록체인에 최적화하기 위해 Move에 적용된 개선 사항은 [Move on Sui](/concepts/sui-move-concepts.mdx#differences) 문서를 참조한다.
