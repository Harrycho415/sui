Sui 계정을 생성하고 CLI 환경을 설정한다.




```sh
$ sui client
```

`sui client` CLI 도구를 처음 실행하는 경우, Sui 전체 노드 서버 URL과 의미 있는 환경 별칭을 입력하라는 메시지가 표시된다. 또한 `sui.keystore` 에 임의의 키 쌍이 포함된 주소와 구성 `client.yaml` 파일이 생성된다.

기본적으로 `client.yaml` 및 `sui.keystore` 파일은 `~/.sui/sui_config` 에 있다. 자세한 내용은 [Sui client CLI tutorial](/references/cli/client.mdx)을 참조한다.

`sui client`를 처음 실행하는 것이 아니라면 로컬 환경에 이미 `client.yaml` 파일이 있을 것이다. 이 튜토리얼을 위해 새 address를 생성하려면 다음 명령어를 사용한다.

```sh
$ sui client new-address ed25519
```
