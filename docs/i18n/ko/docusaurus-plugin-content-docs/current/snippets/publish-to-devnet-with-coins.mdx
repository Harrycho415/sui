Testnet에 패키지를 게시하기 전에 Testnet SUI 토큰이 필요하다. 토큰을 얻으려면 https://faucet.sui.io/ 에서 온라인 faucet을 방문한다. Testnet 계정에서 SUI 토큰을 얻는 다른 방법은 [Get SUI Tokens](/guides/developer/getting-started/get-coins)를 참조한다.

이제 테스트넷 SUI 계정이 있으므로 컨트랙트를 배포할 수 있다. 패키지를 게시하려면 동일한 터미널 또는 콘솔에서 다음 명령을 사용한다:

```
sui client publish --gas-budget <GAS-BUDGET>
```

가스 예산 계산 시에는 `20000000` 과 같은 표준값을 사용한다.
