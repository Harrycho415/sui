<TabItem label="suiup" value="suiup">

먼저 `suiup`을 설치한다:

```sh
$ curl -sSfL \
  https://raw.githubusercontent.com/Mystenlabs/suiup/main/install.sh \
  | sh
```

그다음 Sui를 설치한다:

```sh
$ suiup install sui@testnet
```

대체 설치 방법은 [`suiup` repository](https://github.com/MystenLabs/suiup)를 참고한다.

:::danger

`suiup`으로 Sui를 설치해도 client는 구성되지 않는다. `sui` command를 사용하려면 [configure the Sui client](/guides/developer/getting-started/configure-sui-client.mdx)를 해야 한다.

Sui가 올바르게 설치되었는지 확인하려면:

1. 터미널이나 console을 연다
2. `sui --version`을 입력하고 Enter를 누른다

"command not found" error를 받으면 Sui binary 디렉터리가 `PATH` environment variable에 있는지 확인한다.

:::

</TabItem>

<TabItem label="Homebrew" value="brew">

<TabItem label="Chocolatey" value="choco">

[Chocolately](https://chocolatey.org/)가 설치되어 있어야 다음 command를 실행할 수 있다:

```sh
$ choco install sui
```

Chocolatey community website의 [versions of Sui for Windows](https://community.chocolatey.org/packages/sui)에서 더 많은 version을 찾는다.

Sui가 올바르게 설치되었는지 확인하려면:

1. 터미널이나 console을 연다
