:::caution

운영 환경에서는 공용 엔드포인트 대신 전용 노드/공유 서비스를 사용한다. Mysten Labs에서 관리하는 공용 엔드포인트( `fullnode.<NETWORK>.sui.io:443` )는 요청 수에 제한이 있으며, 30초당 100건의 요청만 지원한다. 트래픽이 많은 프로덕션 어플리케이션에서는 공용 엔드포인트를 사용하지 않는다.

풀 노드를 직접 운영하거나 전문 인프라 제공업체에 위탁할 수 있다(트래픽이 많은 애플리케이션의 경우 후자를 권장한다). [Sui 개발자 포털](https://sui.io/developers#dev-tools) 에서 **'Node Service'** 태그를 사용하여 Sui용 신뢰할 수 있는 RPC 엔드포인트 제공업체 목록을 확인할 수 있다.

:::
