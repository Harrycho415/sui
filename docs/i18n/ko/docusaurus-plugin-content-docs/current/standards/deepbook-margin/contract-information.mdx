## Contract versions

DeepBook Margin은 upgradeable contract를 사용한다. contract가 업그레이드되면 `MARGIN_PACKAGE_ID`만 업데이트하면 된다 - 별도로 언급되지 않는 한 이전 버전은 호환된다.

## Risk parameters

Risk parameter는 각 trading pair에 대한 leverage limit과 liquidation threshold를 결정한다. 이 parameter가 작동하는 방식에 대한 자세한 설명은 [Risk Ratio](./contract-information/risk-ratio.mdx)를 참조한다.

### SUI/USDC (5x leverage)

| Parameter | Value | Description |
|-----------|-------|-------------|
| Min Withdraw Risk Ratio | 2.0 | 담보를 인출하기 위해 필요한 최소 비율 |
| Min Borrow Risk Ratio | 1.25 | 차입하기 위해 필요한 최소 비율 |
| Liquidation Risk Ratio | 1.1 | position이 liquidatable 상태가 되는 비율 |
| Target Liquidation Risk Ratio | 1.25 | liquidation 후 목표 비율 |
| User Liquidation Reward | 2% | liquidator에 대한 보상 |
| Pool Liquidation Reward | 3% | pool에 대한 보상 |
