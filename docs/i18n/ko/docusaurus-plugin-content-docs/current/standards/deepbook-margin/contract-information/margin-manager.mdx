`MarginManager`는 `BalanceManager`를 감싸며 deposit, withdraw, trade, 그리고 leveraged position을 관리하는 데 필요한 capability를 제공하는 shared object이다. 이는 사용자가 margin pool에서 asset을 차입하여 trading position을 확대할 수 있게 하면서, collateralization을 통해 risk를 관리하도록 한다.

각 `MarginManager`는 특정 DeepBook pool과 연관되며, 해당 pool에서 trading을 허용하는 margin pool로부터 차입할 수 있다. margin manager는 차입 position을 추적하고 시스템 건전성을 유지하기 위해 risk ratio limit을 강제한다.
